Features

  • Recieve your own virtual number
    • Virtual numbers are assigned by AWS. We may be able to provide custom numbers upon request.
    • Audio prompts to greet and instruct customers on how to leave requests are customizeable and read by a text-to-speech bot
  • Online access through web and mobile application
    • The application allows you to manage the 'On Call' schedule, view information about current and past requests, and edit company/employee information.
    • The application allows you to manage the 'On Call' schedule, view information about current and past requests, and edit company/employee information.
    • The mobile app provides the same functionality as the web app, and can be installed on a user's home screen.
  • Route Calls During/After Work Hours
    • Choose your own physical number to direct normal calls to, if desired
    • Suited for organizations with regularly-houred office staff
  • Ensure Reliability With a Callback Schedule
    • The OnCall system will try multiple times to call the 'On Call' employee. If the 'On Call' employee fails to answer a set number of calls, the call will be elevated and routed to a manager
  • Receive Transcription and Recording of Request Calls
    • Receive a text message with the audio transcription as well as a secure, time-sensitive link to the audio recording of the call
    • The recording can also be accessed at any time through the OnCall app
    • When listening to the message via phone call, a text-to-speech rendition of the audio transcription is heard. This is currently a limitation of Amazon's Connect system, and something for which we are looking for a workaround
